# Diário de Bordo: Portfólio Daniel Salgado 🚀

> "Eu não apenas construo interfaces. Eu estruturo experiências, fluxos, integrações e produtos digitais que resolvem problemas reais."

Este documento funciona como uma timeline viva do desenvolvimento do meu novo portfólio. A ideia é abandonar o formato tradicional e transformar o site em uma verdadeira **prova de capacidade técnica**, funcionando como um hub de projetos (CMS), laboratório de automações e blog técnico.

---

## 🎯 A Ideia & Fundações
**Objetivo:** Criar um posicionamento profissional moderno. Mostrar que sou um "Builder" (construo E-commerces, SaaS e Ferramentas) focado em código limpo, arquitetura escalável e conversão.

### Por que reestruturar?
- O antigo não refletia minha experiência full-stack / integrações.
- Necessidade de um CMS próprio para classificar os projetos por stack (`React`, `Node`, `Tailwind`) e contexto (`B2B`, `SaaS`, `E-Commerce`).
- Vontade de implementar um pipeline de conteúdo automatizado.

### 🛠 A Stack Escolhida
- **Core:** Next.js (App Router), TypeScript, React.
- **Estilização & UI:** Tailwind CSS (v4), shadcn/ui, Framer Motion (para micro-animações premium e sensação "dark tech").
- **Workflow / IA:** Antigravity Kit (Automação de Agents e Guidelines de código na pasta oculta `.agent`).
